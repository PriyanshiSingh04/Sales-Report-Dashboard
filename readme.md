# Sales Report Dashboard

## Project Overview
This Power BI dashboard provides insights into sales performance, revenue, profit, and product-wise analysis.

## Dashboard Features

- Total Revenue Analysis
- Gross Profit Tracking
- Units Sold Overview
- Revenue by Product
- Revenue by Quarter
- Gross Profit by Month
- Top Sales Representatives
- Country, Year, and Month Filters

## KPIs

- Total Revenue: 126M
- Gross Profit: 87M
- Units Sold: 4M

## Tools Used

- Power BI
- Excel
- DAX
- Data Modeling
